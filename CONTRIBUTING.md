# Contributing to Digital FTE Agent

Thank you for considering contributing to the Digital Full-Time Employee AI Customer Success Agent!

## Development Process

We follow Spec-Driven Development (SDD):

1. **Constitution** - Core principles and constraints
2. **Specification** - Detailed feature requirements
3. **Plan** - Architecture and implementation approach
4. **Tasks** - Actionable work items
5. **Implementation** - Code development

## Getting Started

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Ensure all tests pass
5. Submit a pull request

## Code Standards

- Follow the project constitution
- Write tests before implementation (TDD)
- Keep changes small and focused
- Update documentation as needed
- Follow existing code style

## Reporting Issues

Please use the issue tracker to report bugs or suggest features.
